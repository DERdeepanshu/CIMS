/*************************************************
PREPARING THE SETUP FOR PROPER RUNNING OF THE CODE
/*************************************************

1. Use any Java compatible IDE eg- eclipse.
2. Add the jar file src/collegeManagementSystem/miglayout-3.5.5.jar to the reference library of the IDE
	eg:- In eclipse
		1. Right Click on the Project Folder on the Left side in your IDE.
		2. Click on the Build Path >> Configure Build Path
		3. A window will open their in the libraries tab select the Module Path and click the "Add JARs.." button on the right and choose the file according the above path
		4. Click Apply and ok.
3. Add the jar file src/collegeManagementSystem/mysql-connector-java-8.0.20.jar to the reference library of the IDE following the same steps as step 2.
4. Run the Managementsystem.java file.
